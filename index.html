English: VCC2018[1]

There are 4 systems implemented for comparison: bPPG-LI (baseline): The cross-lingual voice conversion system using bilingual PPG (bPPG) with a single language-independent (LI) output layer. mPPG-LI: The cross-lingual voice conversion system using mixed-lingual PPG (mPPG) with a single language-independent (LI) output layer. bPPG-LS: The cross-lingual voice conversion system using bilingual PPG (bPPG) with language-specific (LS) output layers. mPPG-LS: The cross-lingual voice conversion system using mixed-lingual PPG (mPPG) with language-specific (LS) output layers.

en → cn denotes we are converting an English source utterance to a Mandarin target speaker's voice.
cn → en denotes we are converting a Mandarin source utterance to an English target speaker's voice.
